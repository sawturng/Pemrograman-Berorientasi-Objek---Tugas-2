# Pemrograman Berorientasi Objek — Tugas 1

## Deskripsi
Project ini merupakan implementasi konsep **Percabangan dalam Pemrograman Berorientasi Objek (PBO)** menggunakan bahasa Java.

Program dibuat sebagai bagian dari tugas mata kuliah Pemrograman Berorientasi Objek dan berfokus pada penggunaan:
- Percabangan `if`
- Percabangan `if-else`
- Operator logika (`&&`, `||`, `!`)
- Tipe data `char` dan `int`

---

## Pembahasan Soal 

### 1️⃣ Mengecek Pola Huruf
Program menerima tiga huruf dari pengguna dan mengecek apakah urutannya adalah:
Konsonan – Vokal – Konsonan.

### 2️⃣ Menentukan Kubus atau Bukan
Program menerima tiga nilai sisi dan menentukan apakah bangun tersebut termasuk kubus (semua sisi sama) atau bukan.

### 3️⃣ Menghitung Lama Bekerja
Program menghitung lama bekerja berdasarkan jam masuk dan jam keluar dalam sistem 12 jam menggunakan percabangan.
